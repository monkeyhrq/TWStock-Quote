# 📈 TWStock Hunter (台股終極選股神器)

這是一個基於 Python 與 Tkinter 開發的台股技術分析與自動篩選工具。
透過整合 `yfinance` 與 `twstock`，提供多執行緒的即時盤後數據下載，並支援自訂高彈性的技術指標過濾條件，幫助投資人快速找出符合個人交易邏輯的標的。

## ✨ 核心特色與亮點

* **高度彈性的篩選條件**：支援日線(D)、周線(W)、月線(M)級別的交叉分析。
* **獨創「指標容錯機制」**：實作 KD 指標的數值與次數容錯邏輯（例如：允許 K 值微幅跌破 D 值 2.0 以內），避免因短期雜訊而誤殺長線趨勢向上的優質標的。
* **防追高乖離系統**：內建日、周、月線的均線乖離率（Bias）限制，確保進場點貼近防守均線。
* **Data-Driven 三欄式 UI**：採用極簡三欄式介面與水平捲軸，並透過資料驅動自動生成 UI，避免程式碼冗餘。系統會自動記憶使用者的所有策略參數。
* **內建互動式 K 線圖**：無縫串接 `mplfinance`，點擊篩選結果即可直接呼叫包含成交量、均線、KD、MACD 的技術分析圖表。

## 📂 檔案架構說明

* `stock_yahoo.py`：主程式（包含 GUI 介面、資料下載、技術分析運算與畫圖邏輯）。
* `stock_config.json`：策略設定檔。專案內附一份預設的嚴謹選股範本，程式執行時會自動記憶上次的篩選條件。
* `update_codes.py`：**SSL 連線修復工具**。專門解決台灣證交所 API 阻擋導致 `twstock` 無法更新代碼的問題。
